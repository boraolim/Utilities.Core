GUIA DE PROTECCION DE LIBRERIA (Solo Windows):

1. Instalar .NET Reactor 6.5.0.0 (ya puede compilar aplicaciones en .NET Core 3.1 en adelante).
2. Instalar Nuget Package Manager.
3. Abrir el archivo "PROTECCION.nrproj" en .NET Reactor y proteger la librería.
4. Ejecutar el comando de dotnet para generar el paquete Nuget: dotnet pack Utilities.csproj -p:NuspecFile=nuspec\Utilities.Core.nuspec -o nuspec
5. Abrir Nuget Package Manager e incluir las librerías protegidas en el archivo de Nuget generado.
6. Subir el paquete: dotnet nuget push Utilities.Core.dll.1.0.22.1054.nupkg -s https://www.nuget.org -k oy2ctdxzmte2njkwamnqoo6rf3noytve5jdezjn5f5526y

Signing library:
1. Export public key from the PFX: sn -p key.pfx key.pub
2. Enable delayed signing for your assembly by adding <DelaySign>true</DelaySign> to the project file or by AssemblyDelaySignAttribute. Use a public key from step 1 instead of the PFX for signing.
3. Re-sign your assembly after build: sn -R assembly.dll key.pfx
4. al /out:MyAssembly.dll MyModule.netmodule /keyfile:sgKey.snk
